# think-cog — Iterative thinking partner powered by CellCog

> Complex ideas need iteration, not conversation. Think → Do → Review → Repeat across research, documents, visuals, data, and prototypes.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
think-cog is a complex reasoning partner for problems whose solution emerges through doing. Powered by CellCog SDK, it executes across all modalities in a fire-and-forget async loop — producing real outputs at each iteration. Requires `cellcog` installed first.

## Features
- Think → Do → Review → Repeat loop
- Multi-modal: research, documents, visuals, data, prototypes
- Fire-and-forget async with `notify_session_key`
- CellCog SDK-powered

## Usage / Quick Start
```bash
clawhub install cellcog
```
Describe a complex problem. think-cog runs async and notifies when results are ready.
